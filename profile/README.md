## 🏢 Profile & Vision

**Dream Apps Labs** is an independent software engineering group focused on building highly optimized, privacy-first, on-device mobile applications. Our long-term (LT) product vision is centered on delivering software that operates with absolute privacy, zero-cloud dependency, and elegant user interfaces.

We believe that mobile applications should empower users without compromising their personal data. By executing machine learning models and data storage directly on the device, our apps remain fast, secure, and fully operational offline.

---

## 📱 Flagship Project: Smart Money AI Expense tracker

Our launch product, **Smart Money AI Expense tracker**, is a cognitive personal finance companion engineered for modern mobile architectures.

👉 **[Read the Smart Money AI Privacy Policy](https://dreamappsdevs.github.io/policy/smart-money/privacy-policy.html)**

<details open>
<summary><b>Key Architectural Specifications</b></summary>

* **On-Device Cognitive Classifier**: Integrates a localized **TinyBERT SMS classifier** powered by the **ONNX Runtime** and **TensorFlow Lite** to analyze financial text messages on-device in under 5ms, without uploading data to external servers.
* **Storage Engine**: Built on a reactive **SQLite database layer** via **Jetpack Room**, utilizing coroutine flows for asynchronous data streams.
* **Visual & Styling Systems**: Styled with a premium glassmorphic theme system, customized AMOLED pitch black canvas, micro-interactions, and tabular alignments using monospaced layout systems.
* **Authentication Gate**: Intercepts app entry using native Android Biometric prompts (Fingerprint/PIN/Pattern verification).
</details>

## 🔒 Security & Privacy Compliance

Our development guidelines comply strictly with the highest international security standards:
* **Zero External Calls**: The core data engine runs with zero internet traffic requirements, completely avoiding man-in-the-middle exploits.
* **Data Minimization**: We do not collect, serialize, or transmit any user transaction records, location coordinates, or device IDs.
* **Google Play Policy Alignment**: Built using native SDKs (including official Google Play Billing SDK for commercial features) to guarantee full compatibility and compliance with Google's target API level guidelines.
